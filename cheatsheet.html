The For Loop

  text += "The number is " + i + "<br>";
}

Objects

var person = {
    firstName : "John",
    lastName  : "Doe",
    age       : 50,
    eyeColor  : "blue"
};

When accessing object properties, use dot notation to access explicit object properties. Use bracket notation if using variables for properties.

person.firstName;
"John"

dog["firstName"];
"John"

var x = "age";
person.x;
undefined

person[x];
50

The For/In Loop

var txt = "";
var person = {fname:"John", lname:"Doe", age:25}; 
var x;
for (x in person) {
    txt += person[x] + " ";
}
John Doe 25

The While Loop

var text = "";
var i = 0;
while (i < 10) {
    text += "<br>The number is " + i;
    i++;
}
